#  CutNow

O **CutNow** é um aplicativo de barbearias criado com o objetivo de facilitar o agendamento de horários e a descoberta de novos estabelecimentos.

##  Sobre o projeto

O projeto surgiu a partir da dificuldade que muitas pessoas enfrentam ao tentar marcar horários em barbearias, principalmente no dia a dia corrido.

O CutNow propõe uma solução simples e prática, permitindo que o usuário encontre barbearias, visualize serviços e agende horários de forma rápida.

##  Funcionalidades

* Agendamento de horários
* Listagem de barbearias
* Visualização de serviços
* Escolha de barbeiros
* Notificações de lembrete

##  Objetivo

Facilitar o processo de agendamento em barbearias, tornando-o mais rápido, simples e acessível.
